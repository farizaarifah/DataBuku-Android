# Data Buku Fariza
Data Buku Android adalah suatu aplikasi dari Buku Fariza untuk mendata buku seperti menambah dan melihat list buku mengunakan firebase. dan juga ada aplikasi maps untuk melihat lokasi dimana Buku Fariza itu berada.
## Technology used in this app 
1. Firebase 
2. Location Based Service
3. Intent
4. Picasso
5. Scroll View 
6. Recycler View
7. Floating Button
## How to use
berikut penjelasannya.<br>
**1. Splash screen.** kita perlu menunggu sebentar lalu akan diarahkan ke halaman utama. <br>
<img src="/ss/splash_screen.jpg" width="270" height="585"> <br>
**2. Halaman utama.** Di halaman utama disuguhkan 2 menu yaitu menu data buku dan menu lokasi.
Menu data buku untuk melihat daftar buku apa saja yang ada di Ruang Baca Fariza. <br>
<img src="/ss/halaman_utama.jpg" width="270" height="585"> <br>
Sedangkan menu lokasi untuk melihat lokasi Ruang Baca Fariza dengan tampilan maps. <br>
**3. Halaman list data buku.**
Jika kita memilih menu data buku maka akan diarahkan ke halaman list data buku. Di halaman ini diperlihatkan seluruh buku yang dimiliki Ruang Baca Fariza dalam bentuk list. Tetapi di list ini hanya menampilkan judul buku dan penulis saja. <br> Untuk melihat data buku lebih lanjut, kita bisa memilih salah satu data di list tersebut yang mana akan diarahkan ke halaman detail activity <br>
**4. Halaman detail.** <br> Di halaman ini kita bisa melihat dengan detail informasi dari buku yang dipilih. Dan juga dapat melihat cover buku tersebut. <br> <img src="/ss/detail_buku.jpg" width="270" height="585"> <br>
**5.Tambah data buku.** <br> 
Tambah data buku dapat dilakukan dengan cara mengeklik floating action button yang ada di halaman list data. Floating button berada di pojok kanan bawah yang mana akan mengarahkan kita ke halaman tambah data yang berbentuk form. <br> <img src="/ss/floating_button.jpg" width="270" height="585"> <br>
**6. Halaman tambah data.** <br>
halaman ini berbentuk form ntuk mengisi data buku baru. Data yang harus diisi adalah Judul buku, Penulis, Penerbit, dan cover buku. Di sini akan terjadi upload image, oleh karena itu ada progress bar di bawah ruang untuk gambar. <br>
